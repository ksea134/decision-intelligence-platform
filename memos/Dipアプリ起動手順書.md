# Dipアプリ起動手順書（非エンジニア向け）

> 作成日: 2026-03-21
> 対象者: 非エンジニア（中学生でもわかるレベル）

---

## はじめに：Dipアプリの仕組み

Dipアプリは「2つのプログラム」が協力して動いています。

| 名前 | 役割 | 例えると |
|------|------|---------|
| **バックエンド** | データを管理・処理する | 厨房の料理人 |
| **フロントエンド** | 画面を表示する | ホールの接客係 |

お客さん（あなた）がブラウザで注文（操作）すると、接客係（フロントエンド）が厨房（バックエンド）に伝え、料理（データ）を持ってきます。

**両方が動いていないと、アプリは正しく動きません。**

---

## ステップ0：Macを起動する

1. Mac miniの電源ボタンを押す
2. ログイン画面が出たらパスワードを入力
3. デスクトップが表示されたら準備完了

---

## ステップ1：ターミナルを開く

### ターミナルとは？
Macに「文字で命令を出す」ためのアプリです。マウスでクリックする代わりに、キーボードで命令を打ち込みます。

### 開き方
1. **Cmd + Space** を押す（Spotlightという検索窓が出る）
2. 「**terminal**」と入力
3. **Enter** を押す

黒い画面が出たら成功です。

---

## ステップ2：バックエンドを起動する

### このステップでやること
「厨房」を開店させます。

### 入力するコマンド
以下をコピーして、ターミナルに貼り付けて **Enter** を押す：

```
cd ~/dip && source .venv/bin/activate && uvicorn backend.main:app --reload --port 8000
```

### コマンドの意味（知りたい人向け）

| 部分 | 意味 |
|------|------|
| `cd ~/dip` | 「dip」というフォルダに移動する（cdは「change directory」の略） |
| `source .venv/bin/activate` | Python専用の作業部屋（仮想環境）に入る |
| `uvicorn backend.main:app` | バックエンドを起動するプログラム |
| `--reload` | コードを変更したら自動で反映する |
| `--port 8000` | 「8000番の窓口」で待機する |

### 成功したら
こんな表示が出ます：
```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete.
```

**このターミナルは閉じないでください！** 閉じるとバックエンドが止まります。

---

## ステップ3：新しいターミナルタブを開く

### なぜ新しいタブが必要？
ステップ2のターミナルは「バックエンド専用」になりました。フロントエンド用に別の作業場所が必要です。

### やり方
1. ターミナルがアクティブな状態で
2. **Cmd + T** を押す

新しいタブが開きます。

---

## ステップ4：フロントエンドを起動する

### このステップでやること
「接客係」を配置します。

### 入力するコマンド
新しいタブに以下を貼り付けて **Enter**：

```
cd ~/dip/frontend && npm run dev 2>/dev/null
```

### コマンドの意味（知りたい人向け）

| 部分 | 意味 |
|------|------|
| `cd ~/dip/frontend` | 「frontend」フォルダに移動する |
| `npm run dev` | フロントエンドを開発モードで起動する |
| `2>/dev/null` | エラーメッセージを非表示にする（**重要：これがないとMacがフリーズする可能性あり**） |

### 成功したら
こんな表示が出ます：
```
▲ Next.js 16.2.0 (Turbopack)
- Local:        http://localhost:3000
✓ Ready in 200ms
```

---

## ステップ5：ブラウザでアプリを開く

### やり方
1. ブラウザ（Chrome、Safari等）を開く
2. アドレスバーに以下を入力して **Enter**：

```
http://localhost:3000
```

### 成功したら
「意思決定支援AI」の画面が表示され、左側のメニューから企業を選べます。

---

## 終了するとき

### フロントエンドを止める
フロントエンドのターミナルタブで **Ctrl + C** を押す

### バックエンドを止める
バックエンドのターミナルタブで **Ctrl + C** を押す

### ターミナルを閉じる
**Cmd + Q** でターミナル全体を終了

## トラブルシューティング

### 「ポートが使用中」と出たら

別のプログラムが同じ窓口を使っています。

確認コマンド：
```
lsof -i :8000
```

止めるコマンド（数字はPID、表示された番号に置き換える）：
```
kill 数字
```

### 企業が選べない・データが出ない

バックエンドが起動していないか、ポートが違います。
- バックエンドのターミナルを確認
- `--port 8000` で起動しているか確認

### Macがフリーズした

1. 電源ボタンを **10秒長押し** して強制シャットダウン
2. 再起動後、上記の手順を最初からやり直す
3. フロントエンド起動時は必ず `2>/dev/null` をつける
