| 環境 | URL | 用途 |
|------|-----|------|
| ローカル（バックエンド） | http://localhost:8000 | 開発・デバッグ（FastAPI） |
| ローカル（フロントエンド） | http://localhost:3000 | 開発・デバッグ（Next.js） |
| 本番 (Cloud Run) | https://dip-897403315215.asia-northeast1.run.app | デモ・本番運用 |

## 使い分けの目安

- **普段の開発・修正** → ローカル環境（変更が即反映、エラーも即確認）
- **本番と同じ動きか最終確認** → Cloud Runにデプロイ

## 環境比較

| 観点 | ローカル | Cloud Run |
|------|---------|-----------|
| コード変更の反映 | ◎ 即座 | △ ビルド→デプロイが必要（数分） |
| エラーの確認 | ◎ ターミナルで即表示 | △ Cloud Logging |
| 試行錯誤のしやすさ | ◎ 何度でもすぐやり直せる | △ 毎回デプロイし直し |
| 費用 | ◎ 無料 | △ デプロイのたびにビルド費用 |
| 外出先アクセス | ✗ 不可 | ◎ 可能 |
| 認証 | なし | ◎ IAP |

## Dipアプリの仕組み

Dipアプリは「2つのプログラム」が協力して動いている。

| 名前 | 役割 | 例えると |
|------|------|---------|
| **バックエンド** | データを管理・処理する | 厨房の料理人 |
| **フロントエンド** | 画面を表示する | ホールの接客係 |

ブラウザで操作すると、接客係（フロントエンド）が厨房（バックエンド）に伝え、データを持ってくる。**両方が動いていないと、アプリは正しく動かない。**

## 起動手順（ローカル）

### ステップ1: ターミナルを開く

Macに「文字で命令を出す」ためのアプリ。

1. **Cmd + Space** を押す（Spotlight検索窓が出る）
2. 「**terminal**」と入力
3. **Enter** を押す

黒い画面が出たら成功。

### ステップ2: バックエンドを起動する

以下をコピーしてターミナルに貼り付けて **Enter**：

```
cd ~/dip && source .venv/bin/activate && uvicorn backend.main:app --reload --port 8000
```

| 部分 | 意味 |
|------|------|
| `cd ~/dip` | 「dip」フォルダに移動する |
| `source .venv/bin/activate` | Python仮想環境に入る |
| `uvicorn backend.main:app` | バックエンドを起動する |
| `--reload` | コード変更を自動反映する |
| `--port 8000` | 8000番ポートで待機する |

成功すると以下が表示される：
```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete.
```

**このターミナルは閉じないこと。** 閉じるとバックエンドが止まる。

### ステップ3: 新しいターミナルタブを開く

ターミナルがアクティブな状態で **Cmd + T** を押す。

### ステップ4: フロントエンドを起動する

新しいタブに以下を貼り付けて **Enter**：

```
cd ~/dip/frontend && npm run dev 2>/dev/null
```

| 部分 | 意味 |
|------|------|
| `cd ~/dip/frontend` | 「frontend」フォルダに移動する |
| `npm run dev` | フロントエンドを開発モードで起動する |
| `2>/dev/null` | エラーメッセージを非表示にする（**重要：つけないとMacがフリーズする可能性あり** #012参照） |

成功すると以下が表示される：
```
▲ Next.js 16.2.0 (Turbopack)
- Local:        http://localhost:3000
✓ Ready in 200ms
```

### ステップ5: ブラウザでアプリを開く

ブラウザ（Chrome、Safari等）で以下を開く：
```
http://localhost:3000
```

「意思決定支援AI」の画面が表示され、左側のメニューから企業を選べる。

## 終了するとき

1. フロントエンドのターミナルタブで **Ctrl + C**
2. バックエンドのターミナルタブで **Ctrl + C**
3. **Cmd + Q** でターミナル全体を終了

## トラブルシューティング

### 「ポートが使用中」と出たら
```
lsof -i :8000   # 使用中のプロセスを確認
kill [PID]       # プロセスを停止（PIDは表示された番号に置き換える）
```

### 企業が選べない・データが出ない
- バックエンドのターミナルを確認
- `--port 8000` で起動しているか確認（**8080ではない！**）

### Macがフリーズした
1. 電源ボタンを **10秒長押し** して強制シャットダウン
2. 再起動後、上記の手順を最初からやり直す
3. フロントエンド起動時は必ず `2>/dev/null` をつける

## クイックリファレンス（コピペ用）

```
# バックエンド起動
cd ~/dip && source .venv/bin/activate && uvicorn backend.main:app --reload --port 8000

# フロントエンド起動（別タブで）
cd ~/dip/frontend && npm run dev 2>/dev/null

# ブラウザURL
http://localhost:3000
```
